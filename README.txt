Project: Notre Dame Monopoly - "DuLacpoly" (Irishopoly was copyrighted :[ )

Collaborators: Michael Creehan, Victor Hernandez, Jodi Lo, Kyle Mulholland

Requirements: SDL library along with the SDL.ttf library.

  DuLacopoly is an SDL version of the popular monopoly game with a Notre Dame
theme. At the beginning of the game, the user is prompted with a menu with
various options.
